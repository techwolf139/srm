# 中国合同审查技能改进方案

## 现状分析

当前技能库中**缺乏专门的法律合同审查技能**，尤其是针对中国法律体系的合同审查能力。

---

## 需要新增的核心技能

### 1. **中国合同法基础审查技能** (`china-contract-review`)

**功能定位：**
- 审查合同是否符合《中华人民共和国民法典》合同编的规定
- 验证合同必备条款的完整性
- 识别违反强制性法律规定的条款

**核心能力：**
```
- 合同主体资格审查（民事行为能力、授权委托）
- 合同形式要件审查（书面形式、签字盖章）
- 必备条款检查（当事人、标的、数量、质量、价款、履行期限等）
- 无效条款识别（违反法律强制性规定、损害社会公共利益）
- 可撤销条款识别（重大误解、欺诈、胁迫、显失公平）
```

**法律依据：**
- 《中华人民共和国民法典》合同编
- 《最高人民法院关于适用〈民法典〉合同编通则若干问题的解释》

---

### 2. **商事合同专项审查技能** (`commercial-contract-review`)

**功能定位：**
针对常见商事合同类型的专业化审查

**支持的合同类型：**
```
1. 买卖合同 - 所有权转移、风险承担、质量保证
2. 租赁合同 - 租赁期限、租金支付、维修责任
3. 借款合同 - 利息约定、担保措施、还款计划
4. 承揽合同 - 工作成果交付、验收标准、报酬支付
5. 技术合同 - 知识产权归属、保密义务、技术成果分享
6. 劳动合同 - 试用期、工资支付、社会保险、解除条件
7. 合伙合同 - 出资份额、利润分配、债务承担
8. 保证合同 - 保证方式、保证期间、保证范围
```

---

### 3. **合同风险识别技能** (`contract-risk-assessment`)

**功能定位：**
系统性识别和评估合同中的法律风险

**风险评估维度：**
```
【高风险】
- 违约责任不明确或过低
- 争议解决条款缺失或不利
- 合同解除条件不对等
- 知识产权归属模糊
- 保密义务范围过宽或过窄
- 不可抗力定义不合理

【中风险】
- 履行期限约定不明
- 质量标准不具体
- 付款条件不清晰
- 通知送达地址缺失

【低风险】
- 格式条款未显著提示
- 文字表述不够严谨
- 附件引用不完整
```

**输出要求：**
- 风险等级评定（高/中/低）
- 风险条款原文标注
- 修改建议及法律依据
- 替代条款示例

---

### 4. **合同条款优化技能** (`contract-clause-optimization`)

**功能定位：**
提供符合中国法律实践的标准条款和优化建议

**核心模块：**
```
1. 违约责任条款库
   - 违约金计算方式（固定金额/比例/实际损失）
   - 迟延履行责任
   - 根本违约认定标准

2. 争议解决条款库
   - 仲裁条款（仲裁机构选择、仲裁规则）
   - 诉讼管辖条款（被告住所地/合同履行地/原告住所地）
   - 涉外合同管辖选择

3. 知识产权保护条款
   - 权利归属约定
   - 许可使用范围
   - 侵权责任承担

4. 保密条款模板
   - 保密信息定义
   - 保密期限
   - 例外情形
   - 违约责任

5. 不可抗力条款
   - 事件范围界定
   - 通知义务
   - 证明责任
   - 后果处理
```

---

### 5. **行业特定合同审查技能** (`industry-specific-contract-review`)

**功能定位：**
针对不同行业的监管要求进行专项审查

**行业分类：**
```
【金融行业】
- 银行贷款合同 - 利率合规、担保登记
- 保险合同 - 免责条款提示说明义务
- 证券交易合同 - 投资者适当性

【房地产行业】
- 商品房买卖合同 - 预售许可、面积差异处理
- 建设工程合同 - 资质要求、竣工验收
- 物业服务合同 - 收费标准、服务标准

【互联网行业】
- 用户服务协议 - 个人信息保护、格式条款
- 数据合作协议 - 数据安全、跨境传输
- 平台入驻协议 - 平台责任、商家管理

【医疗行业】
- 医疗服务合同 - 知情同意、医疗风险
- 药品采购合同 - GSP 合规、质量追溯

【外贸行业】
- 进出口合同 - 贸易术语、报关责任
- 国际货物买卖 - CISG 适用、信用证条款
```

---

### 6. **合同履约监控技能** (`contract-performance-monitoring`)

**功能定位：**
跟踪合同履行状态，预警履约风险

**监控要点：**
```
- 关键时间节点提醒（付款日、交货日、验收日）
- 履约证据收集指导（送货单、验收报告、对账单）
- 违约情形识别与应对建议
- 诉讼时效管理（3 年时效期间计算）
- 合同变更管理（补充协议、书面确认）
```

---

### 7. **电子合同合规技能** (`e-contract-compliance`)

**功能定位：**
审查电子合同的法律效力和合规性

**审查要点：**
```
- 电子签名有效性（CA 认证、时间戳）
- 数据电文形式要求（可调取查用）
- 第三方平台资质（电子合同服务平台备案）
- 存证保全措施（区块链存证、公证存证）
- 隐私政策合规（个人信息保护法）
```

**法律依据：**
- 《电子签名法》
- 《电子商务法》
- 《个人信息保护法》
- 《网络安全法》

## 技能实现的技术要求

### 1. **法律知识库建设**

```
【法律法规层】
- 法律（全国人大及其常委会制定）
- 行政法规（国务院制定）
- 司法解释（最高人民法院）
- 部门规章（各部委）
- 地方性法规

【案例数据库】
- 指导性案例（最高人民法院发布）
- 公报案例
- 典型裁判案例
- 各地高院参考案例

【实务指引】
- 合同示范文本（市场监管总局等发布）
- 行业协会指引
- 律师实务操作指南
```

---

### 2. **审查流程标准化**

```
Step 1: 合同类型识别
        ↓
Step 2: 主体资格核查
        ↓
Step 3: 形式要件审查
        ↓
Step 4: 实质内容审查
        ↓
Step 5: 风险等级评估
        ↓
Step 6: 修改建议生成
        ↓
Step 7: 审查报告输出
```

---

### 3. **输出格式规范**

**审查报告应包含：**
```markdown
# 合同审查意见书

## 一、合同基本信息
- 合同名称
- 合同各方
- 签订日期
- 审查日期

## 二、审查依据
- 适用的法律法规
- 参考的案例

## 三、审查发现

### （一）合法性问题
[列出违反法律强制性规定的条款]

### （二）完整性问题
[列出缺失的必备条款]

### （三）风险提示
[按风险等级分类列出]

## 四、修改建议
[逐条给出具体修改意见和法律依据]

## 五、结论
[综合审查结论]

## 六、声明
[免责声明：本审查意见仅供参考，不构成正式法律意见]
```

## 与中国法律实践的结合点

### 1. **民法典时代的审查要点**

《民法典》2021 年施行后的新变化：
```
- 保证方式推定变更（一般保证→连带责任保证）
- 流质条款效力缓和
- 情势变更原则明确化
- 预约合同独立地位
- 保理合同有名化
- 物业服务合同有名化
- 合伙合同有名化
```

### 2. **司法实践的倾向性**

```
【格式条款审查】
- 提供方需尽到提示说明义务
- 免除自身责任、加重对方责任的条款易被认定无效

【违约金调整】
- 超过实际损失 30% 可能被调减
- 需举证证明实际损失

【合同解除】
- 催告程序的必要性
- 合理期限的认定标准
```

### 3. **地域差异考量**

```
- 各地高院对同类问题的裁判尺度可能不同
- 经济特区、自贸区的特殊政策
- 民族自治地方的变通规定
```

## 实施建议

### 第一阶段：基础能力建设（1-2 个月）
1. 创建 `china-contract-review` 核心技能
2. 建立民法典合同编知识库
3. 实现基础条款审查功能

### 第二阶段：专业化扩展（2-3 个月）
1. 开发商事合同专项审查模块
2. 建立风险识别模型
3. 完善条款优化建议库

### 第三阶段：行业深化（3-6 个月）
1. 针对重点行业开发专用审查规则
2. 整合案例数据库
3. 实现智能风险评分

### 第四阶段：生态整合（持续）
1. 与飞书文档深度集成
2. 支持合同版本对比
3. 实现履约提醒功能

---

## 注意事项

### ⚠️ 法律边界
```
1. 明确定位为"辅助工具"，非替代律师
2. 所有输出需包含免责声明
3. 复杂案件建议咨询专业律师
4. 不涉及诉讼策略建议
```

### ⚠️ 数据合规
```
1. 合同内容属于敏感信息，需加密存储
2. 不得将客户合同用于模型训练
3. 遵守《数据安全法》《个人信息保护法》
4. 境内数据存储，不出境
```

### ⚠️ 持续更新
```
1. 法律法规修订及时同步
2. 新司法解释及时纳入
3. 典型案例定期收录
4. 审查规则动态优化
```

---

## 总结

要使技能体系符合中国合同审查要求，需要：

1. **新增 7 大核心技能**覆盖合同审查全流程
2. **建立法律知识库**确保审查依据准确
3. **标准化审查流程**保证审查质量一致
4. **结合中国法律实践**体现本土化特色
5. **重视合规边界**明确工具定位

这将是一个系统工程，需要法律专业知识与 AI 技术的深度融合。
